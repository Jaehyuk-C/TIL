# branch

## 1. branch 생성

```bash
$ git branch <브랜치이름>
```



## 2. branch 이동

```bash
$ git checkout <브랜치이름>
```



## 3. branch 생성 및 이동

```bash
$ git checkout -b <브랜치이름>
```



## 4. branch 조회

```bash
$ git branch
```



## 5. branch merge

``` bash
(master) $ git merge <브랜치이름>
```



## 6. branch 삭제

``` bash
$ git branch -d <브랜치이름>
```
